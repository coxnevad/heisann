Handles the logic behind the stop button. If the stop button is pushed, the function will:
1. Set the elevator direction to "stopped", and stop the motor
2. Turn on the stop light
3. Delete the Elevator queue
Then it returns 1 if the button was pushed, or 0 if not.

Parameters
[out]elevator_queueThe array that represents the queue system of the elevator.
[out]stop_array_upThe array that indicates if the elevator floor panel up buttons are pushed. This will be reset.
[out]stop_array_downThe array that indicates if the elevator floor panel down buttons are pushed. This will be reset.
[out]elevator_panel_lights_arrayThe array that indicates if the elevator panel lights are on. This will be reset
[out]directionThe variable that represents the direction of the elevator.
Returns
int of value "1" or "TRUE" if the stop button is pushed, "0" or "FALSE" if it is not pushed.

[in,out]ref_timeThe variable that controls the time of the elevator door. It is used here to make sure that it takes 3 seconds from opening the elevator door untill it closes.
[out]directionA variable that gives the current direction of the elevater.

void stopbutton_flag_discarder ( int *  stop_button_flag,
int *  floor_sensor 
)

This is supposed to fix a bug that we had related to the stop button and stopping between two floors. The function is used each loop.
If the stop button flag is high and the floor sensor is in an undefined state, then we set the stop button flag low, in order to avoid a bug where the elevator gets stuck between two floors.

Parameters
[in,out]stop_button_flagA stop flag to indicate if the stop button has been pushed. It has to be reset in order for the bug to be fixed.
[in]floor_sensorA variable that shows what floor state the elevator is in. This is to indicate if the elevator is between two floors.
