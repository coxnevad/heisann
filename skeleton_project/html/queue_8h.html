Deletes orders of the elevator queue, light array or the stop arrays when the elevator stops at a floor.
The arrays are handled by setting the indexed floor in the array to zero.
Deletes the first element of the elevator queue, then sorts the queue.

Parameters
[in]floorA variable that shows the current floor of the elevator.
[out]elevator_queueAn array that represents the queue system of the elevator.
[out]stop_array_upAn array that indicates if the elevator floor panel up buttons are pushed.
[out]stop_array_downAn array that indicates if the elevator floor panel down buttons are pushed.
[out]elevator_panel_lights_arrayAn array that indicates if the elevator panel lights are on.
